Componentes base

Átomos compartidos entre pista usuario y pista admin. Definiciones técnicas en componentes-base.md.

Button — variantes

7 variantes según contexto. Cada una existe en 4 tamaños (xl, lg, md, sm). Ver componentes-base.md sección 1.

Button — tamaños

XL para sticky CTAs mobile · LG para CTAs en cards · MD default mobile · SM admin denso.

Badge — variantes

Píldoras de estado para chips, tags, status indicators.
★ Mejor cuota Verificada MINCETUR Activa Pendiente Cancelada ● EN VIVO 💎 Premium ⏱ Cierra en 8 min

Avatar — tamaños

Iniciales sobre fondo dorado cuando no hay foto. 5 tamaños xs/sm/md/lg/xl.

Card — variantes

Default · Elevated · Urgent · Premium · Flat. Padding y radius parametrizables.
Card default
Background blanco con borde sutil y sombra-sm. Uso: contenedor estándar.
Card urgent
Para partidos cercanos a cerrar (countdown crítico).

Background oscuro con borde dorado. Solo Premium.
